# Notational

> Notational Velocity for the web built with [Vue.js](https://vuejs.org/).

![Notational screenshot](screenshot.png)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# deploy on Firebase
npm run deploy
```
